# Tezrisat Frontend

This is the React + TypeScript frontend for Tezrisat.

## Development

```bash
npm install
npm run dev
```

The app runs at `http://localhost:5173`.

## Build

```bash
npm run build
```

## Notes

- Backend API URL can be configured via `VITE_API_URL` in `.env`.
- See the root `docs/setup.md` for full setup steps.
